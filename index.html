JUEGO DE SNAKE

Desarrollo de un juego tipo Snake utilizando JavaScript. Implementa un game loop, manipulación dinámica de arrays para el crecimiento del personaje, detección de colisiones con límites y sistema de puntuación actualizado en tiempo real mediante manipulación del DOM.

Este proyecto fue desarrollado con el objetivo de fortalecer mis habilidades en manipulación del DOM, la implementación de lógica de colisiones y el manejo dinámico de estructuras de datos como arrays, aplicados en un entorno interactivo en tiempo real.

Este proyecto fue desarrollado con ayuda de documentacion MDN, chatgpt, conocimiento previo.
