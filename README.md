# Portafolio Web - Andres Rivera

Este es mi portafolio web personal desarrollado con HTML, CSS y JavaScript puro.

## Características

- Diseño moderno y responsive
- Navbar adaptable (modo hamburguesa en móvil)
- Animaciones con CSS
- Sección "Acerca de mí"
- Juego interactivo tipo Snake desarrollado con JavaScript
- Sistema de puntuación en tiempo real
- Manipulación dinámica del DOM
- Lógica de colisiones y estructuras de datos (arrays)

## Proyecto destacado

El juego Snake fue desarrollado implementando:
- Game loop
- Detección de colisiones con bordes
- Crecimiento dinámico del personaje
- Actualización de puntuación en tiempo real

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
